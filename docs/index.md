# Local-first "Circle to Search" desktop app for Windows, macOS, and Linux

Squigit lets you capture anything on screen and ask AI about it instantly—without sending your data through a middleman server.

## Quick links

- **Repository:** [github.com/a7mddra/squigit](https://github.com/a7mddra/squigit)
- **Latest release:** [Download Squigit](https://github.com/a7mddra/squigit/releases/latest)
- **Documentation:** [Project docs](./README.md)

## Core highlights

- ⚡ Global hotkey capture (`Super/Cmd/Win + Shift + A`)
- 🧠 Multi-model AI analysis (Gemini)
- 🔍 One-click Google Lens handoff
- 🔐 BYOK + local-first architecture
